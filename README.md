# Bucket List App

**write by expressjs and vuejs**

## Project setup
```
npm install
```

### Compiles Server and hot-reloads for development
```
npm run server
```

### Compiles Client and hot-reloads for development
```
npm run client
```

### Compiles both Server and client 
```
npm run dev
```
